# Tushare 配置指南

本工具现已支持 Tushare 接口获取A股数据，相比 Yahoo Finance 更适合获取中国股票实时行情。

## 获取 Tushare Token

1. 访问 [Tushare官网](https://tushare.pro) 注册账号
2. 在个人中心获取 API Token
3. 复制你的 Token（格式如：`1a2b3c4d5e6f7g8h9i0j...`）

## 配置方式

### 方式一：环境变量（推荐）

在系统环境变量中添加：
```
TUSHARE_TOKEN=你的token
```

或在 PowerShell 中临时设置：
```powershell
$env:TUSHARE_TOKEN="你的token"
```

### 方式二：.env 文件

1. 复制 `.env.example` 为 `.env`：
```bash
copy .env.example .env
```

2. 编辑 `.env` 文件，填入你的 Token：
```
TUSHARE_TOKEN=你的token
```

### 方式三：启动时传入

在启动脚本中临时设置：
```powershell
$TUSHARE_TOKEN="你的token"
python start.py
```

## A股代码格式

使用A股代码时支持以下格式：

- **带交易所后缀**：`000001.SZ`（平安银行）、`600000.SH`（浦发银行）
- **纯6位数字**：`000001`、`600000`（程序会自动判断交易所）

| 000001.SZ | 平安银行 | 深交所 |

| 000858.SZ | 五粮液 | 深交所 |

## 验证配置

启动后端服务后，查看控制台输出：
```
[INFO] Tushare API 初始化成功
```

如果看到此消息，说明配置成功。

## 故障排查

### 问题：未初始化 Tushare
**现象**：控制台显示 `[INFO] 未配置 Tushare Token，跳过初始化`

**解决**：
1. 检查环境变量是否正确设置
2. 确保 `.env` 文件位于项目根目录
3. 重启后端服务

### 问题：API 限流
**现象**：`api限流，请稍后再试`

**解决**：
1. Tushare 免费用户有积分限制，建议升级会员
2. 或者使用模拟数据模式（会自动降级）

### 问题：股票数据不存在
**现象**：`无法获取 xxx 的数据`

**解决**：
1. 检查股票代码格式是否正确
2. 确认该股票在 Tushare 中有数据
3. 尝试使用带交易所后缀的完整代码

## 美股和港股

如果未配置 Tushare Token 或查询的是非A股代码，程序会自动回退到 **Yahoo Finance**：

- **美股**：`AAPL`、`TSLA`、`MSFT`
- **港股**：`0700.HK`（腾讯）、`BABA`（阿里）

## 推荐配置

开发测试时使用 Tushare（A股数据更准确）：
```
TUSHARE_TOKEN=你的token
```

生产环境建议同时保留两个数据源作为备份。
