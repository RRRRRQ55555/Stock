# 页面模式参考

24 种落地页和仪表板模式。

## 落地页模式

### 1. Hero-Centric Design

**核心特点**: 视觉驱动的 Hero 区域
**适用场景**: 视觉识别强的产品、SaaS、应用
**页面结构**:
```
1. Hero Section (60% viewport)
   - 大标题 + 副标题
   - 主 CTA
   - 产品截图/插图
2. Logo Cloud (客户标志)
3. Feature Highlights
4. Testimonials
5. CTA Section
6. Footer
```

**关键元素**:
- 全宽 Hero 背景（渐变/图片/视频）
- 大标题（48-64px）
- 主 CTA 按钮（突出对比色）
- 社交证明元素

### 2. Conversion-Optimized

**核心特点**: 转化驱动设计
**适用场景**: 潜在客户生成、销售页面
**页面结构**:
```
1. Above-the-fold CTA
2. Problem Statement
3. Solution Overview
4. Features/Benefits
5. Social Proof (Stats + Testimonials)
6. Pricing/CTA
7. FAQ
8. Final CTA
```

**关键元素**:
- 多个 CTA 位置
- 利益导向文案
- 紧迫性元素（限时优惠）
- 信任标识（安全认证、客户数）

### 3. Feature-Rich Showcase

**核心特点**: 详细功能展示
**适用场景**: SaaS、复杂产品
**页面结构**:
```
1. Hero
2. Quick Overview (3-4 核心功能)
3. Detailed Features (逐一展开)
4. Use Cases/Workflows
5. Integrations
6. Testimonials
7. CTA
```

**关键元素**:
- 功能分类展示
- 截图/演示视频
- 动画演示
- 对比表格

### 4. Minimal & Direct

**核心特点**: 极简、直奔主题
**适用场景**: 简单产品、应用、工具
**页面结构**:
```
1. Hero (标题 + CTA)
2. 3 Steps/Features
3. Simple Pricing
4. CTA
5. Footer
```

**关键元素**:
- 大量留白
- 简洁文案
- 单一焦点
- 快速转化

### 5. Social Proof-Focused

**核心特点**: 社会证明为核心
**适用场景**: 服务、B2C 产品、新品牌
**页面结构**:
```
1. Hero
2. Trust Indicators (客户数、评分)
3. Testimonials Carousel
4. Case Studies
5. Media Mentions
6. Team/About
7. CTA
```

**关键元素**:
- 客户评价轮播
- 数据统计展示
- 媒体报道标识
- 案例研究

### 6. Interactive Product Demo

**核心特点**: 产品演示为核心
**适用场景**: 软件、工具、复杂产品
**页面结构**:
```
1. Hero + "Try Demo" CTA
2. Interactive Demo Section
3. Key Features
4. How It Works
5. Use Cases
6. Testimonials
7. Final CTA
```

**关键元素**:
- 可交互演示
- 产品截图/视频
- 功能导览
- 免费试用入口

### 7. Trust & Authority

**核心特点**: 建立信任和权威性
**适用场景**: B2B、企业、咨询
**页面结构**:
```
1. Hero (专业定位)
2. Expertise/Team
3. Services
4. Case Studies
5. Awards/Certifications
6. Thought Leadership (Blog/Publications)
7. Contact CTA
```

**关键元素**:
- 团队介绍
- 资质认证
- 专业内容
- 案例研究

### 8. Storytelling-Driven

**核心特点**: 品牌故事叙事
**适用场景**: 品牌、机构、非营利
**页面结构**:
```
1. Story Hook (情感共鸣)
2. Problem Journey
3. Solution Story
4. Values/Mission
5. Impact/Results
6. Team/Community
7. Join Us CTA
```

**关键元素**:
- 情感化文案
- 视觉叙事
- 品牌价值观
- 社区/影响展示

## 仪表板模式

### 1. Data-Dense Dashboard

**核心特点**: 高密度数据展示
**适用场景**: 复杂数据分析、监控
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Header (搜索、通知、用户)              │
├──────────┬──────────────────────────────┤
│          │                              │
│ Sidebar  │   Main Content Area          │
│ (导航)   │   ┌─────┐ ┌─────┐ ┌─────┐   │
│          │   │ KPI │ │ KPI │ │ KPI │   │
│          │   └─────┘ └─────┘ └─────┘   │
│          │   ┌─────────────────────┐   │
│          │   │    Main Chart       │   │
│          │   └─────────────────────┘   │
│          │   ┌─────────┐ ┌─────────┐   │
│          │   │ Detail  │ │ Detail  │   │
│          │   └─────────┘ └─────────┘   │
└──────────┴──────────────────────────────┘
```

**关键元素**:
- 多 KPI 卡片
- 主图表区域
- 详细数据表格
- 筛选器

### 2. Executive Dashboard

**核心特点**: 高层决策视图
**适用场景**: C-level、管理摘要
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Header (日期范围、导出)                │
├─────────────────────────────────────────┤
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐    │
│  │ KPI │  │ KPI │  │ KPI │  │ KPI │    │
│  └─────┘  └─────┘  └─────┘  └─────┘    │
├─────────────────────────────────────────┤
│  ┌───────────────────────────────────┐  │
│  │         Trend Chart              │  │
│  └───────────────────────────────────┘  │
├────────────────┬────────────────────────┤
│  Top Items     │   Alerts/Actions         │
│  ┌──────────┐  │   ┌────────────────┐   │
│  │ List     │  │   │ Action Items   │   │
│  │          │  │   │                │   │
│  └──────────┘  │   └────────────────┘   │
└────────────────┴────────────────────────┘
```

**关键元素**:
- 关键指标突出
- 趋势图表
- 行动项列表
- 简洁布局

### 3. Real-Time Monitoring

**核心特点**: 实时监控视图
**适用场景**: 运维、DevOps、IoT
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Status Bar (系统状态、警报数)          │
├─────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────────┐   │
│  │ Real-time   │  │ Live Metrics    │   │
│  │ Chart 1     │  │ Grid            │   │
│  └─────────────┘  └─────────────────┘   │
├─────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────────┐   │
│  │ Real-time   │  │ Alert Feed      │   │
│  │ Chart 2     │  │ (auto-scroll)   │   │
│  └─────────────┘  └─────────────────┘   │
├─────────────────────────────────────────┤
│  System Logs (collapsible)              │
└─────────────────────────────────────────┘
```

**关键元素**:
- 实时数据更新
- 警报系统
- 状态指示器
- 日志面板

### 4. Drill-Down Analytics

**核心特点**: 深入分析能力
**适用场景**: 数据探索、BI 工具
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Header (筛选器、日期范围)              │
├──────────┬──────────────────────────────┤
│          │                              │
│ Filter   │   Breadcrumb Navigation      │
│ Panel    │   ┌─────────────────────┐   │
│ (多层级) │   │    Main Chart        │   │
│          │   └─────────────────────┘   │
│          │   ┌─────────────────────┐   │
│          │   │    Sub-chart         │   │
│          │   └─────────────────────┘   │
│          │   ┌─────────────────────┐   │
│          │   │    Data Table        │   │
│          │   │   (drill-down rows)  │   │
│          │   └─────────────────────┘   │
└──────────┴──────────────────────────────┘
```

**关键元素**:
- 面包屑导航
- 分层筛选
- 钻取图表
- 详细表格

### 5. Comparative Analysis

**核心特点**: 并排比较视图
**适用场景**: A/B 测试、竞品分析
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Comparison Controls (选择对比项)       │
├──────────────────────┬──────────────────┤
│      Item A          │      Item B      │
├──────────────────────┼──────────────────┤
│  ┌──────────────┐   │  ┌──────────────┐│
│  │   Header     │   │  │   Header     ││
│  │   (名称/图)  │   │  │   (名称/图)  ││
│  └──────────────┘   │  └──────────────┘│
├──────────────────────┼──────────────────┤
│  ┌──────────────┐   │  ┌──────────────┐│
│  │   Metrics    │   │  │   Metrics    ││
│  └──────────────┘   │  └──────────────┘│
├──────────────────────┼──────────────────┤
│  ┌──────────────┐   │  ┌──────────────┐│
│  │   Chart      │   │  │   Chart      ││
│  └──────────────┘   │  └──────────────┘│
├──────────────────────┼──────────────────┤
│  ┌──────────────┐   │  ┌──────────────┐│
│  │   Details    │   │  │   Details    ││
│  └──────────────┘   │  └──────────────┘│
└──────────────────────┴──────────────────┘
```

**关键元素**:
- 并排布局
- 差异高亮
- 对比指标
- 切换控制

### 6. Predictive Analytics

**核心特点**: 预测和建议
**适用场景**: 预测分析、ML 洞察
**布局结构**:
```
┌─────────────────────────────────────────┐
│  Model Controls (模型选择、参数)        │
├─────────────────────────────────────────┤
│  ┌───────────────────────────────────┐  │
│  │     Prediction Chart              │  │
│  │     (历史 + 预测区间)             │  │
│  └───────────────────────────────────┘  │
├────────────────────┬────────────────────┤
│  ┌──────────────┐  │  ┌──────────────┐  │
│  │ Confidence   │  │  │ Key Drivers  │  │
│  │ Metrics      │  │  │ (影响因子)   │  │
│  └──────────────┘  │  └──────────────┘  │
├─────────────────────────────────────────┤
│  Recommendations Panel                  │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐     │
│  │ Rec 1   │ │ Rec 2   │ │ Rec 3   │     │
│  └─────────┘ └─────────┘ └─────────┘     │
└─────────────────────────────────────────┘
```

**关键元素**:
- 预测图表
- 置信区间
- 驱动因素
- 行动建议

│  │ Chart        │  │  │ Curve        │   │

**关键元素**:
- 漏斗图
- 留存曲线
- 用户旅程
- 热力图

**关键元素**:
- 资产卡片
- 投资组合图
- 绩效图表
- 交易记录

## 响应式布局原则

### 桌面端 (≥1024px)
- 侧边栏导航
- 多列网格布局
- 完整数据展示

### 平板端 (768px - 1023px)
- 收起侧边栏（汉堡菜单）
- 2 列网格
- 简化图表

### 移动端 (<768px)
- 底部导航栏
- 单列布局
- 卡片堆叠
- 关键指标优先

## 模式选择指南

| 目标 | 推荐模式 |
|------|---------|
| 产品展示 | Hero-Centric, Feature-Rich |
| 转化优化 | Conversion-Optimized, Minimal & Direct |
| 建立信任 | Social Proof-Focused, Trust & Authority |
| 品牌叙事 | Storytelling-Driven |
| 数据监控 | Real-Time Monitoring, Data-Dense |
| 决策支持 | Executive Dashboard, Predictive Analytics |
| 用户分析 | User Behavior Analytics |
| 财务管理 | Financial Dashboard |
