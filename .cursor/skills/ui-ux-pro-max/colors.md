## 快速选择指南

### 按情绪选择

| 情绪 | 推荐配色 |
|------|---------|
| 专业可信赖 | 科技蓝、海军蓝 |
| 活力创新 | 琥珀金、紫罗兰 |
| 平静放松 | teal、鼠尾草绿 |
| 奢华高端 | 黑+金、深蓝+金 |
| 友好温暖 | 柔粉、浅琥珀 |
| 大胆现代 | 橙红、亮紫 |

### 按行业选择

| 行业 | 推荐配色 |
|------|---------|
| 科技/SaaS | 科技蓝、天空蓝 |
| 金融 | 深蓝+金、海军蓝 |
| 医疗 | 青蓝、teal |
| 电商 | 热情粉、活力橙 |
| 创意 | 紫+粉、多彩 |
| 教育 | 天蓝、琥珀 |

### 配色验证清单

- [ ] Primary 与 Background 对比度 ≥ 4.5:1
- [ ] Text 与 Background 对比度 ≥ 4.5:1
- [ ] CTA 颜色在背景上突出
- [ ] Secondary 与 Primary 协调
- [ ] 整体配色符合品牌情绪
- [ ] 考虑色盲用户可识别性
