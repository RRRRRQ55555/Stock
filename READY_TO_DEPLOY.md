### 步骤 2️⃣：连接 Vercel

1. 打开 https://vercel.com
2. 点击 "Sign Up" → 选择 "GitHub"
3. 授权并登录

### 步骤 3️⃣：导入并部署

1. 点击 "Add New" → "Project"
2. 选择 `stock-assistant` 仓库
3. 填写配置：
   ```
   Root Directory: frontend
   Build Command: npm run build
   Output Directory: dist
   ```
4. 点击 **"Deploy"**
